# Extension Vue Template
## ✨ FEATURES

✔ Vue as front-end
✔ Tailwind 
✔ Testing  
✔ browserExtension plugin

# Local development

## Project setup

```
yarn install
```

## Compiles and hot-reloads for development

```
yarn dev
```

## Compiles and minifies for production

```
yarn build
```

## Lints and fixes files

```
yarn lint
```
